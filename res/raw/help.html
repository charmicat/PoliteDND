Creating a schedule:

 1 - Select the + icon
2 - Select a Start Time
3 - Select an End Time (or select Never for a "Manual Stop" schedule. It will run until manually disabled)
4 - Select the days of the week which the schedule should be repeated (or leave unselected for a "One-shot" schedule, which will run only once at the same day it was created)


Schedules are created enabled by default. To toggle on/off individual schedules, select the Calendar button.


You may close the interface upon configuration. A background service will check the time, start/stop schedules as needed and notify you of such events.
However, schedules will not be checked if the app is disabled. Select the "Disabled" button to enable it.
To disable checking, select the "Enabled" button.
