Creating a schedule:

1 - Select the icon
2 - Select a Start Time
3 - Select an End Time (or select Never for a "Manual Stop" schedule)
4 - Select the days of the week which the schedule should be active (or leave unselected for a "One-shot" schedule)

Schedules are created enabled by default. To toggle on/off individual schedules, select the "ON"/"OFF" button.

You may close the interface upon configuration. A background service will check the time, start/stop schedules as needed and notify you of such events.

Schedules will not be checked if the app is disabled. Select the "Disabled" button to enable it.
To disable checking, select the "Enabled" button.
